## Internationalization

### i18next 25.6.0

**Comprehensive i18n solution:**

```json
{
  "dependencies": {
    "i18next": "^25.6.0",
    "react-i18next": "^16.1.3"
  }
}
```

**Supported Languages:**
- 🇬🇧 English (en)
- 🇪🇸 Spanish (es)
- 🇫🇷 French (fr)

**Implementation Example:**

```javascript
import { useTranslation } from "react-i18next";

function Component() {
  const [t] = useTranslation("global");
  
  return (
    <div>
      <h1>{t("home.title")}</h1>
      <p>{t("home.description")}</p>
    </div>
  );
}
```

**Features:**
- **Namespace support** for organized translations
- **Lazy loading** of translation files
- **Pluralization** and interpolation
- **Route translation** for SEO-friendly URLs
- **LocalStorage persistence** for language preference

<Accordion title="Translation File Structure">
  ```
  src/locales/
  ├── en/
  │   └── global.json
  ├── es/
  │   └── global.json
  └── fr/
      └── global.json
  ```
  
  Each language has its own directory with translation JSON files organized by namespace.
</Accordion>
